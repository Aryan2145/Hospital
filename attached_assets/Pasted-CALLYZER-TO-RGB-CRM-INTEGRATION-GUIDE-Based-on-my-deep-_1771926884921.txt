CALLYZER TO RGB CRM INTEGRATION GUIDE
Based on my deep study of both platforms, here's your complete step-by-step integration guide for pulling call data, employee data, and master data from Callyzer to your RGB CRM using webhook API.

OVERVIEW: INTEGRATION ARCHITECTURE
Your integration will work through these three main data flows:

CALL DATA → From Callyzer Call Log APIs to RGB CRM

EMPLOYEE DATA → From Callyzer Employee APIs to RGB CRM

MASTER DATA → Configuration data from Callyzer to RGB CRM

PART 1: UNDERSTANDING CALLYZER API FUNDAMENTALS
Key Technical Details from Callyzer
​
API Base URL: https://api1.callyzer.co/api/v2.1/

Rate Limiting: One request per 2 seconds (429 Too Many Requests if exceeded)

Authentication: Bearer token in Authorization header

Format: Authorization: Bearer <YOUR_API_TOKEN>

Content Type: All requests must be application/json

PART 2: GETTING YOUR CALLYZER API TOKEN

Navigate to Connectors > API & Webhook > API Config

Enter your Callyzer login credentials

Click Generate Token

CRITICAL: Save this token securely - you'll need it for all API calls to Callyzer.

Callyzer sends data via HTTP POST to your RGB CRM webhook endpoint

RGB CRM receives the data in JSON format

Your CRM system processes and stores the data

STEP 4: Implement Webhook Receiver in RGB CRM
Your RGB CRM backend should have code like this (pseudo-code for different languages):

Python (Flask) Example:

python
from flask import Flask, request, jsonify
import hmac
import hashlib

app = Flask(__name__)
CALLYZER_SECRET = "your_webhook_secret_key"

@app.route('/api/integrations/callyzer', methods=['POST'])
def receive_callyzer_webhook():
    # Validate webhook signature
    signature = request.headers.get('X-Callyzer-Signature')
    body = request.get_data()
    
    expected_sig = hmac.new(
        CALLYZER_SECRET.encode(),
        body,
        hashlib.sha256
    ).hexdigest()
    
    if signature != expected_sig:
        return jsonify({"error": "Invalid signature"}), 401
    
    # Parse webhook data
    data = request.json
    
    # Process each employee's call logs
    for employee_group in data:
        emp_number = employee_group['emp_number']
        emp_name = employee_group['emp_name']
        
        # Get or create employee in RGB CRM
        employee = Employee.get_or_create(
            phone=emp_number,
            name=emp_name
        )
        
        # Process each call
        for call in employee_group['call_logs']:
            # Get or create lead
            lead = Lead.get_or_create(
                phone=call['client_number'],
                name=call['client_name']
            )
            
            # Create call record
            call_record = CallLog.create(
                employee=employee,
                lead=lead,
                call_type=call['call_type'],
                duration=call['duration'],
                call_datetime=f"{call['call_date']} {call['call_time']}",
                notes=call['note'],
                recording_url=call['call_recording_url'],
                status=call.get('crm_status'),
                callyzer_id=call['id']
            )
            
            # Link to Callyzer lead if available
            if 'lead_id' in call:
                call_record.callyzer_lead_id = call['lead_id']
                call_record.save()
    
    return jsonify({"status": "success", "message": "Data received"}), 200

STEP 6: For EMPLOYEE DATA (Pull Approach)
Unlike webhooks (push), you may want to PULL employee data on a schedule.

Scheduled Pull Method:

python
import requests
from datetime import datetime, timedelta

CALLYZER_API_TOKEN = "your_api_token"
CALLYZER_BASE_URL = "https://api1.callyzer.co/api/v2.1"

def sync_employees_from_callyzer():
    """Run this daily via cron job"""
    
    headers = {
        "Authorization": f"Bearer {CALLYZER_API_TOKEN}",
        "Content-Type": "application/json"
    }
    
    # Fetch all employees
    response = requests.get(
        f"{CALLYZER_BASE_URL}/employee/get",
        headers=headers,
        params={
            "page_no": 1,
            "page_size": 100
        }
    )
    
    if response.status_code == 200:
        employees = response.json()['result']
        
        for emp in employees:
            # Create or update employee in RGB CRM
            Employee.create_or_update(
                phone=emp['emp_number'],
                code=emp['emp_code'],
                name=emp['emp_name'],
                country_code=emp['emp_country_code'],
                tags=emp.get('emp_tags', []),
                app_version=emp.get('app_version'),
                lead_active=emp.get('is_lead_active'),
                recording_active=emp.get('is_call_recording_active'),
                last_call_at=emp.get('last_call_at'),
                device_info=emp.get('device_details')
            )
STEP 7: For CALL HISTORY DATA (Pull Approach)
If you want historical call data, use the /call-log/history endpoint:

Sample API Request:

python
import requests
import time
from datetime import datetime, timedelta

def fetch_call_history(days_back=7):
    """Fetch last 7 days of call history"""
    
    headers = {
        "Authorization": f"Bearer {CALLYZER_API_TOKEN}",
        "Content-Type": "application/json"
    }
    
    # Calculate timestamps (must be UNIX in UTC)
    now = datetime.utcnow()
    past = now - timedelta(days=days_back)
    
    call_to_timestamp = int(now.timestamp())
    call_from_timestamp = int(past.timestamp())
    
    payload = {
        "call_from": call_from_timestamp,
        "call_to": call_to_timestamp,
        "page_no": 1,
        "page_size": 100
    }
    
    response = requests.post(
        f"{CALLYZER_BASE_URL}/call-log/history",
        headers=headers,
        json=payload
    )
    
    if response.status_code == 200:
        calls = response.json()['result']
        
        for call in calls:
            # Process each call
            process_call_record(call)
    
    # RESPECT RATE LIMIT: Wait 2 seconds between requests
    time.sleep(2)
